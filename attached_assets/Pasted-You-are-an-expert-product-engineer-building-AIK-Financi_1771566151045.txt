========================
CORE DEFINITIONS (UI WORDING)
========================
Party balance display:
- balance > 0 => “They owe us”
- balance < 0 => “We owe them”
- balance = 0 => “Settled”
Auto-flips on overpay/overreceive.

Working Capital (as defined):
Working Capital = Cash Now + Money Coming − Money Owed
- Cash Now = sum of cash accounts balances
- Money Coming = sum of positive party balances
- Money Owed = sum(abs(negative party balances))

Courier Balance display (per courier party):
- courier_balance > 0 => “Courier owes us” (COD pending)
- courier_balance < 0 => “We owe courier” (charges pending)
- =0 => Settled
